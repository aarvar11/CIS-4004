Student & Course Information System (Java)

Project Image

Description

This project is a Java program that manages student and course information. It loads data from a file, stores students and courses in custom list classes, and supports exporting student data to a JSON file.

Skills Learned

  • Object-oriented programming (classes, inheritance, polymorphism)
  • File input and output
  • Working with custom data structures
  • Manual JSON formatting (no libraries)
